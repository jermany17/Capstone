## 피부르르: AI를 활용한 피부 질환 및 피부 미용 통합 관리 서비스

AI를 활용하여 피부 질환과 피부 미용을 통합적으로 관리할 수 있는 웹 애플리케이션

### 역할
Backend  

### 사용 기술  
Java, Spring Boot, MySQL, JawsDB, AWS S3, Heroku  

### 구현 기능  
- 사용자 인증 기능
- 피부 질환 진단 기능
- 얼굴 피부 미용 진단 기능
- 스킨 케어 미션 기능
- 커뮤니티 기능
